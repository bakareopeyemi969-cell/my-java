DIFFERENT BETWEEN PRIMITIVE AND NON-PRIMITIVE DATA TYPE

Feature
Primitive
Data Types 		Non-Primitive
(Reference)
Data Types
Storage Stored by value in the stack memory. Stored by reference (memory address) in the heap.
Mutability Immutable: Values cannot be changed once created. Mutable: Properties can be added, deleted, or changed.
Comparison Compared by their actual values. Compared by their reference (address) in memory.

Examples of Data Types

  • Primitive Types: JavaScript has 7 primitive types:
    string, number, boolean, undefined, symbol, bigint.
  • Non-Primitive Types: These are essentially Objects. This category includes generic objects {}, Arrays [], Functions, and specialized objects like Date.
