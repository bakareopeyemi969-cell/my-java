15 REVERSED JAVASCRIPT KEYWORD

list of reversed java script keyword: 


1- const: declares a variable that cannot be Reassigned
(a constant) 
2-let: Declares a varariable that can be reassigned
(block-scooped)
3-var: Declares a variable 
 (older method, function-scoped).
4-function: Defines a reusable block of code.
5-if: Starts a conditional statement, executing code only if a condition is true.
6-else: Provides an alternative block of code if the if condition is false.
7-return: Stops a function and sends a value back.
8-for: Creates a loop that runs a specific number of times.
9-while: Creates a loop that runs as long as a condition is true.
10-class : Used to define a blueprint for creating object
 (OOP)
11-new:creates a new instance of an object or class
12-import : Used to bring in functionality from other files 
(modules).
13-export : Used to share functionality with other files 
(modules).
