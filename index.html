The history of javascript

Java script was created in 1995 by Bredan EICH while he was working at Netscape Communications. it was originally developed in just 10 days and first appeared in the Netscape Navigator browser.

The language was intially called Mocha,then changed to LiveScriptand finally named java script developers

in 1997, JavaScript was standardized under the name ECMAScript by ECMA international Since then, it has involved through several versions,adding features like classes,modules,and arrow functions.

Today, javaScript power the web- used for front-end framework like React,Angular,and even on the server with Node,js.
